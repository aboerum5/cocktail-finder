## App Description
Cocktail Finder allows users to choose a liquor or ingredient and view a cocktail possibility using TheCocktailDB API. Users can save cocktail ideas to a favorites list, which is stored and retrieved using MongoDB.

## APIs Information
The app uses TheCocktailDB API to retrieve cocktail data based on the ingredient selected by the user.
